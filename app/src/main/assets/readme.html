一些说明

想使用该App有什么要求？

你必须拥有Xposed框架，并激活该模块，然后重启。假如重启后仍无法生效，那么再重启一下。
假如你就是不能用的话，那抱歉我也没办法了。假如你没有Xposed框架，也希望使用这样的功能，推荐酷友的一款类似应用：用默认浏览器打开链接-酷安网

该App能支持哪些应用？

能支持的App一共分为两部分：一种是我自己维护的，一种是用户自定义的。
我打算维护的App为：QQ各系列、微信、微博、贴吧。因为这些App使用场景比较多，用户量也多。但是比如知乎这些其实也是支持的，只不过得有需要的用户自行添加了。

有些是我遗漏了，有些是存在一些问题。请浏览完本页面的“当前问题”部分，如果没有解释，请把详细的比如哪个页面、什么样的链接，最好有截图。这些信息评论告诉我。（目前会固定看Google Play与酷安的评论）>

屏蔽的应用里有些不该用外置浏览器打开的页面被迫打开了？

这种情况希望你可以反馈给我，自己也可以通过“添加白名单”功能解决，详情请看“使用方法”部分

使用方法

如何自定义屏蔽应用？

一共分成三步：首先点“添加应用”，输入应用的包名，随后重启手机；重启后打开本App，把提示里的链接放入要屏蔽的App中（发消息、发帖等），然后打开它；如果成功，则会自动出现“步骤三”的页面，重启手机即可。

应用的包名是什么？为什么我不可以选择应用？

我相信有能力使用Xposed的用户应该能解决这个问题。至于为什么没做自动选择应用功能，因为这写起来比较烦，我还是希望把精力放在核心代码上。

步骤二里我打开链接了，可是不出来“步骤三”的页面啊？/我步骤都完成了，但是还是无效啊？

有些App代码实在是写的比较奇葩，请把具体情况评论反馈给我。在反馈之前我希望你能确认下不是自己的问题，比如没重启什么的。我收到反馈之后会自己试试的，假如兼容不了我也会回复你的。

是不是在完成步骤后一定得重启手机？

其实不重启也是可以的，但是必须得杀掉屏蔽应用的后台。考虑到国产应用们的流氓，所以还是说重启，防止杀不干净。

这个页面我可以用外置浏览器打开了，但是另一个页面却不行？

实现比较规范的应用比如QQ是只有一个内置浏览器页面的，但是比如贴吧微博这种我就不忍心说了，请尝试长按需要修改的App，选择“添加规则”。这种情况也希望你能反馈给我，因为我并不使用贴吧微博，有些页面我没发现。

“添加规则”是干什么的，如何使用？

“添加规则”就是解决上面这个问题的，使用步骤与屏蔽应用相同。

有些不该被外置浏览器打开的页面被打开了，怎么办？

这是因为有些App的某些页面其实还是用网页实现的，比如QQ的空间部分，留言部分跟情侣空间两个点击都会在本App的1.1版本用外置浏览器打开。我采用了一种简单粗暴的方法：这些链接都是应用的页面（非用户产生），所以直接忽略对应的域名就好了，比如QQ我自带忽略了h5.qzone.qq.com、微博我忽略了card.weibo.com。
使用方法：长按对应App，选择“添加白名单”，添加即可。

QQ的群公告里的链接没法用外置浏览器打开？

这个没法解决，因为整个群公告页面都是处在内置浏览器之中。

微博打开B站的链接还是内置浏览器？

因为B站的分享都采用了share.acg.tv的链接，而这个链接又不会被B站App解析，所以还是得先打开外置浏览器，再打开B站App，因此还不如用内置浏览器得了。
总而言之，是B站的锅，不怪我。
当然，其他一些用了微博分享SDK的应用也是如此，区别就在它们的链接是有名字的，而直接的链接是只有“网页链接”四个字。这就是微博的另一个恶心的地方了。等下个版本看我能否解决这个问题吧。
